# Fluxo ideal do jogo

1- Verifica se o jogador atual tem jogadas válidas.

2- Se não tiver, termina e declara vencedor o adversário.

3- Imprime tabuleiro.

4- Lê e valida movimento.

5- Aplica movimento.

6- Lê e valida remoção.

7- Aplica remoção.

8- Verifica se o adversário ficou sem jogadas.

9- Se ficou, termina com vitória do jogador atual; senão chama play recursivamente com jogador alternado.
